# 2. Prepare data array
Our `$data` in this example is a array and it have following keys:
```php
$data = array(
    'Document' => /*...*/,
    'Supplier' => /*...*/,
    'Buyer'    => /*...*/
);
```
## Linear object notation.
Keys `Document`, `Supplier`, `Buyer` - is a linear object keys
```php
$data = array(
    'Document' => array(
        'name' => 'Bill',
        'number' => '123/A9',
        'date' => '2016-09-23',
        'manager' => ''
    ),
    'Supplier' => array(
        'name' => '',
        ''
    ),
    'Buyer'
);
```
